## iCloud Verification

Before using any reference, verify it's a real file and not an iCloud placeholder:

```bash
file /path/to/reference.png
# Good: "PNG image data, 2040 x 2048, 8-bit/color RGB, non-interlaced"
# Bad:  "EDEADLK" or "Resource deadlock avoided"
```

If placeholder, trigger download: `qlmanage -p /path/to/reference.png >/dev/null 2>&1 && sleep 8`
